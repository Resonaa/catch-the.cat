# catch-the.cat

![logo](./logo.jpg)

A game featuring a black cat inspired by cat figures in Catalan folk tales.

## Building

```sh
$ [npm/yarn/pnpm/bun] install
$ [npm/yarn/pnpm/bun] run build
```
